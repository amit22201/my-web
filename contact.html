Description "My Website" is a personal project designed and developed for the purpose of learning and gaining hands-on experience in web development. The website leverages modern technology stacks, utilizing Spring Boot as the backend framework and Angular as the frontend framework. Major Features Scrollable Grid:- It uses AG-Grid library to show backend data in which when user scrolls it gets data from backend. Data visualisation:- It uses Chart.js library to visualise data from backend in form of chart. Technology Used Back End:- Java, SpringBoot; Data Base:- MySql; Front End:- Angular, CSS, Typescript, HTML; External Library:- Ag-Grid, Chart.js; Developer Name Random Kumar (random@gmail.com)
